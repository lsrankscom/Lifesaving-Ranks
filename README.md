# LSRanks – Lifesaving Results & Rankings

Diese Website zeigt internationale Ergebnisse und Ranglisten im Rettungssport.

## Erste Schritte

1. Öffne [https://github.com/new](https://github.com/new) → neues Repository erstellen  
2. Lade **diesen Ordner** (`lsranks-results-v2_2`) hoch  
3. Gehe zu [https://vercel.com](https://vercel.com) → "Add New Project"  
4. Wähle dein Repo → Deploy  
5. In Vercel → Settings → Domains → `lsranks.com` hinzufügen  
6. Bei [https://ionos.de](https://ionos.de) → DNS-Einstellungen anpassen:
   - A-Record: @ → 76.76.21.21  
   - CNAME: www → cname.vercel-dns.com  
7. Speichern → fertig ✅

## Mehrsprachigkeit

Die Seite ist automatisch in Deutsch, Englisch, Italienisch, Spanisch, Französisch und Niederländisch verfügbar.

## Upgrade

Du kannst später die erweiterte Version (mit Datenbank, Upload, Splits, Auto-Import) einfach nachrüsten.
